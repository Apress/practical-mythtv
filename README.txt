Thanks for taking the time to download the source code for "Practical MythTV:
Building a PVR and Media Center PC". You should check for updates to this code
at apress.com and mythtvbook.com if you intend to run this code on your MythTV
system.

The code is broken into directories by chapter. The contents are:

Chapter03: a simple init script to start the mythbackend process on boot

Chapter06: patches required to get nuvexport working

Chapter12: an updated init script which also starts the DVD transcoding daemon

Chapter13: code for the MythTV instant messaging bot
